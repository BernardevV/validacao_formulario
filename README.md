# Validacao de Formulario

Projeto simples de validacao de formulario usando HTML, CSS e JavaScript puro.

## O que faz
- Verifica se todos os campos estao preenchidos
- Valida email basico (contem @ e .)
- Verifica senha com no minimo 6 caracteres
- Mostra mensagens de erro e sucesso
- Limpa o formulario apos envio correto

## Tecnologias
- HTML
- CSS
- JavaScript

## Como funciona
O formulario impede o envio padrao e faz validacoes basicas no JavaScript.  
Se algo estiver errado, mostra uma mensagem em vermelho.  
Se estiver tudo certo, mostra mensagem de sucesso em verde.
